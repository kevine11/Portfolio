1 Overview

Before we start, lets take a look on the folder structure of the theme. All theme files are placed in the HTML folder. The main HTML folder consists of 3 more folders for diferent types of theme assets:
  1. images - contaning all images for the theme design
  2. js -containing all scripts for the jquery effects
  3. css - containing all CSS files for the theme
  4. twitter - contain php file for twitter configuration

2 Installation

This is an HTML theme and doesn't require instalation in the traditional meaning of the word. You only need to place the content (HTML files and the three folders) in your prefered web directory Keep in mind that the folder structure and file/folder names are important for the correct theme work
